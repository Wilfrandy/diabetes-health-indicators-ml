# Diabetes Risk Prediction (Machine Learning)

Machine learning project that predicts diabetes risk using health and lifestyle indicators from the CDC Diabetes Health Indicators dataset.

## Overview

Built and compared multiple classification models to identify key predictors of diabetes status (No Diabetes / Prediabetes / Diabetes).

## Models

- Random Forest
- Neural Network

## Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- TensorFlow  
- Jupyter Notebook  

## Focus Areas

- Data preprocessing and feature scaling  
- Multi-class classification  
- Model evaluation (Accuracy, F1-score, Confusion Matrix)  
- Feature importance analysis
